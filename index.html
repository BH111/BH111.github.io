About my website

This is my first experience with web design (and pretty much any kind of coding). I have used HTML, CSS and Javascript to create this website. The slideshow at the top of the page is one of the main features - it uses HTML for the actual content, Javascript to change the slides and the dots at the bottom and CSS for the fading animations, styles (e.g. fonts) and positioning of the different parts.
Also try resizing your window - the menu will turn into a drop-down menu and the heading font size will decrease.


After I complete each project I will add to its page and insert a picture in the slideshow.
